# XFS Directory Data Fork

## 一、概述

### 1.1 目录数据fork的作用

目录数据fork存储目录项（directory entries），包含文件名到inode号的映射关系。

```
+----------------------------+
|       目录inode           |
+----------------------------+
|  Data Fork (目录内容)      |
|                            |
|  +-----------------------+ |
|  | 目录块1               | |
|  | - 目录项1: 文件A → ino| |
|  | - 目录项2: 文件B → ino| |
|  +-----------------------+ |
|                            |
|  +-----------------------+ |
|  | 目录块2               | |
|  | - 目录项3: 文件C → ino| |
|  +-----------------------+ |
+----------------------------+
```

### 1.2 目录格式类型

| 格式 | 说明 | 适用场景 |
|------|------|----------|
| 短格式 | 传统线性目录 | 小目录 |
| 块目录 | 块状目录格式 | 中等大小目录 |
| 叶目录 | B+树叶节点 | 大目录 |
| 节点目录 | B+树内部节点 | 超大目录 |

## 二、目录结构格式

### 2.1 短格式目录（Short Format）

用于小目录，目录项直接存储在inode的data fork中：

```
struct xfs_dir2_sf_hdr {
    __u8     count;        /* 目录项数量 */
    __u8     i8count;      /* 8字节inode号计数 */
    xfs_dir2_sf_off_t parent; /* 父目录inode偏移 */
};

struct xfs_dir2_sf_entry {
    __u8     namelen;      /* 名称长度 */
    __u8     offset[2];    /* 偏移量 */
    __u8     name[];       /* 文件名 */
    /* 后面跟着inode号 */
};
```

### 2.2 块目录格式（Block Format）

目录项存储在数据块中：

```
+-----------------------------------+
| 目录块 (通常4KB)                  |
+-----------------------------------+
| 块头 (xfs_dir2_block_hdr)         |
|  - magic: XFS_DIR2_BLOCK_MAGIC    |
|  - bestfree[3]: 空闲空间信息      |
+-----------------------------------+
| 目录项条目 (xfs_dir2_data_entry)  |
|  - inumber: inode号              |
|  - namelen: 名称长度              |
|  - name: 文件名                   |
+-----------------------------------+
| 未尾 (xfs_dir2_data_unused)       |
|  - freetag: 空闲标记              |
|  - length: 空闲长度               |
+-----------------------------------+
```

### 2.3 B+树目录格式

#### 2.3.1 叶节点（Leaf）
```
+-----------------------------------+
| 叶节点块                          |
+-----------------------------------+
| 叶节点头 (xfs_dir2_leaf_hdr)      |
|  - magic: XFS_DIR2_LEAF1_MAGIC    |
|  - count: 条目数量                |
|  - stale: 失效条目数              |
+-----------------------------------+
| 哈希值数组 (__be32 hashval[])     |
+-----------------------------------+
| 目录项地址数组 (__be16 address[]) |
+-----------------------------------+
| 目录项数据区域                    |
+-----------------------------------+
```

#### 2.3.2 节点（Node）
```
+-----------------------------------+
| 节点块                            |
+-----------------------------------+
| 节点头 (xfs_da_node_hdr)          |
|  - magic: XFS_DA_NODE_MAGIC       |
|  - count: 子节点数量              |
+-----------------------------------+
| B+树节点条目 (xfs_da_node_entry)  |
|  - hashval: 哈希值                |
|  - before: 子节点指针             |
+-----------------------------------+
```

## 三、目录B+树结构

### 3.1 B+树索引原理

XFS目录使用B+树进行高效的文件名查找：

```
                     [根节点]
                        |
                 +------+------+
                 |             |
              [内部节点]    [内部节点]
                 |             |
            +----+----+   +----+----+
            |         |   |         |
         [叶节点]  [叶节点] [叶节点] [叶节点]
            |         |   |         |
        目录项块   目录项块 目录项块 目录项块
```

### 3.2 哈希函数

目录项通过哈希函数分布到B+树中：

```c
xfs_dir2_hashname(const unsigned char *name, int namelen)
```

### 3.3 目录操作复杂度

| 操作 | 线性目录 | B+树目录 |
|------|----------|----------|
| 查找 | O(n)     | O(log n) |
| 插入 | O(n)     | O(log n) |
| 删除 | O(n)     | O(log n) |

## 四、目录操作流程

### 4.1 目录查找流程

```
1. 计算文件名哈希值
   ↓
2. 根据目录格式选择查找路径
   ↓
3. 短格式 → 线性扫描
   ↓
4. B+树格式 → 树查找
   ↓
5. 找到目录项 → 返回inode号
   ↓
6. 未找到 → 返回ENOENT
```

### 4.2 目录项插入流程

```
1. 查找插入位置
   ↓
2. 检查是否有足够空间
   ↓
3. 空间不足 → 扩展目录
   ↓
4. 可能触发格式转换：
   - 短格式 → 块格式
   - 块格式 → B+树格式
   ↓
5. 插入新目录项
   ↓
6. 更新父目录链接计数
```

### 4.3 目录项删除流程

```
1. 查找要删除的目录项
   ↓
2. 标记为未使用 (unused)
   ↓
3. 更新最佳空闲空间信息
   ↓
4. 如果删除导致空间浪费：
   - 可能触发格式降级
   - 可能合并目录块
   ↓
5. 更新父目录链接计数
```

### 4.4 目录遍历流程

```
1. 获取目录格式信息
   ↓
2. 短格式 → 顺序读取
   ↓
3. B+树格式 → 中序遍历
   ↓
4. 读取每个目录项
   ↓
5. 过滤"."和".."条目
   ↓
6. 返回目录项列表
```

## 五、内核源码参考

### 5.1 主要数据结构

- `xfs_dir2_sf_hdr`: 短格式目录头
- `xfs_dir2_block_hdr`: 块目录头
- `xfs_dir2_leaf_hdr`: 叶节点头
- `xfs_da_node_hdr`: B+树节点头
- `xfs_dir2_data_entry`: 目录数据条目
- `xfs_dir2_data_unused`: 未使用目录空间

### 5.2 关键函数

- `xfs_dir_lookup()`: 目录查找
- `xfs_dir_createname()`: 创建目录项
- `xfs_dir_removename()`: 删除目录项
- `xfs_dir_replace()`: 替换目录项
- `xfs_dir_cilookup()`: 大小写不敏感查找

### 5.3 文件位置

- `fs/xfs/xfs_dir2.h`: 目录结构定义
- `fs/xfs/xfs_dir2_block.c`: 块目录操作
- `fs/xfs/xfs_dir2_leaf.c`: 叶目录操作
- `fs/xfs/xfs_dir2_node.c`: B+树节点操作
- `fs/xfs/xfs_dir2_sf.c`: 短格式目录操作
